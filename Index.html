  • Drug Information Search:Users can search for medications and receive detailed guidance on usage,dosage, and possible side effects.
  • Symptom Checker: A basic diagnostic tool that predicts potential illness basedon user-entered symptoms.

University Financial Management System

A web based application developed using java servelets,aimed in streamlining and managing the financial operations within a University environment.Key functionalities include:

  • Student Fee management: Tracks payments,balances, and generate receipts.
  • Departmental budgeting: Allows departments to submit, track, and manage budgets.
  • Expenditure Tracking: Records and monitors University expenditures across various departments.
  • Reporting Module: Generates detailed financial reports for administrative review.
  • Role-based Access: secures access for admins, finance officers, and departmental heads.
Built with java servelets, JSP, and MYSQL, the system emphasizes efficiency,trancparency,and automation in university financial workflows.
